# 方向 2: 集成建模

**Integrated Modeling**

---

## 📖 概述

集成建模整合多种建模方法，构建统一的预测框架，是连接数据处理与可靠性分析的核心环节。

## 📂 子方向

### 2.1 基础建模方法
回归模型、分类模型、传统统计建模方法

### 2.2 模型集成方法
集成学习、模型融合、多模型协同

### 2.3 基础模型库
预训练模型、模型库、模型标准化

### 2.4 时序预测方法
时间序列分析、序列建模、动态系统预测

### 2.5 基于深度学习的物理场建模方法
物理信息神经网络、深度学习代理模型、混合建模
