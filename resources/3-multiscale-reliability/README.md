# 方向 3: 多尺度可靠性

**Multiscale Reliability Analysis**

---

## 📖 概述

多尺度可靠性分析研究复杂系统在不同时间和空间尺度下的可靠性评估方法，涵盖异常检测、故障诊断、寿命预测等关键任务。

## 📂 子方向

### 3.1 异常检测方法
统计过程控制、机器学习异常检测、时序异常检测

### 3.2 贝叶斯压缩建模方法
贝叶斯推断、变分推断、模型压缩

### 3.3 故障诊断方法
故障检测、故障分类、根因分析

### 3.4 寿命预测方法
剩余寿命预测、退化建模、可靠性预测
